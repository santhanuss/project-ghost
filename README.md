## 🔬 Technical Details

### Behavioral Authentication
- **Architecture**: LSTM Autoencoder (64→32→32→64)
- **Features**: 9 behavioral biometrics
  - Mouse: speed, variance, acceleration, tremor frequency (8-12 Hz)
  - Keyboard: flight time, dwell time, cognitive pauses
- **Detection**: 95%+ accuracy, <0.1% false positive rate
- **Uniqueness**: Hand tremor patterns cannot be faked by AI

### Security Benefits
- Detects credential theft within 5 minutes
- Immune to deepfake authentication
- Continuous verification (not just login)
- AI-resistant biometrics
